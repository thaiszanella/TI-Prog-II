# TI-Prog-II

API REST para gerenciamento de agendamentos de uma empresa de sonorização, trabalho integrador da disciplina de Programação II.

## Testando a API

O arquivo `TI-Prog-II.yaml` contém a coleção completa do Insomnia com todos os endpoints configurados.

Para importar: abra o Insomnia -> **Import** -> selecione o arquivo `TI-Prog-II.yaml`.

## Tecnologias

- Node.js
- Express

## Instalação

```bash
npm install
```

## Execução

```bash
# Produção
npm start

# Desenvolvimento (hot reload)
npm run dev
```
